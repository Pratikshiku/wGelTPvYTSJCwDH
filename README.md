## Online Tutoring System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/387e534987b8485c9b7578ce2f08b68f/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Forum
5. Open Courses
6. Courses
7. Faculty
8. Personal Center
9. Forum Management
10. Open Course Management
11. Course Management
12. Course Comment Management
13. Course Favorite Management
14. Course Registration Management
15. Faculty Management
16. Carousel Information


**Administrator Functions:**
1. Login
2. Personal Center
3. Open Course Type Management
4. Course Type Management
5. Forum Management
6. Open Course Management
7. Course Management
8. Course Comment Management
9. Course Favorite Management
10. Course Registration Management
11. Faculty Management
12. User Management
13. Carousel Information
